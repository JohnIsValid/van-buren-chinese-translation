## 项目简介

- 本项目旨在将《辐射》系列被取消的续作——**“范布伦计划”（Van Buren）** ——的设计文档及相关资料翻译成中文。
- 范布伦计划是由黑岛工作室（Black Isle Studios）在《辐射2》之后开发的《辐射3》代号。虽然该项目最终于2003年被取消，但其设计文档、区域模板、角色设定、任务脚本等资料已在网络上流传多年，成为《辐射》系列传说中最珍贵的“失落遗产”。后来的《辐射：新维加斯》以及许多《辐射》粉丝作品都从这些设计中汲取了大量灵感。

## 项目目标

- **完整翻译**：将现存所有范布伦计划设计文档翻译成中文
- **术语统一**：建立一套准确、一致的《辐射》世界观术语翻译标准
- **可读性**：在保留原始设计意图的前提下，使中文读者获得流畅的阅读体验
- **开放协作**：欢迎所有《辐射》爱好者参与贡献
