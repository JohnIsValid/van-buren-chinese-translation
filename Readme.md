# 范布伦计划中文翻译

> 致敬永远的黑岛。
>
> WAR, WAR NEVER CHANGES...
## 项目简介

- 本项目致力于将《辐射》系列未完成续作“范布伦计划”（Van Buren）的现存设计文档及相关资料翻译为简体中文。
- “范布伦计划”是黑岛工作室（Black Isle Studios）在《辐射 2》之后开发的《辐射 3》项目代号。项目于 2003 年取消，但其区域设计、人物设定、任务脚本和其他开发资料此后陆续流传。
- 本仓库保存原始英文资料、中文译本、术语规范和项目专属翻译工具，以便长期整理与协作。

## 项目目标

- **完整翻译**：逐步整理并翻译现存的范布伦计划文档。
- **术语统一**：维护准确、一致且可追溯的专有名词译法。
- **忠实原意**：保留原文中的事实、设计意图、任务条件和人物特征。
- **保证可读性**：在不增删原意的前提下，使译文自然、清晰并符合简体中文表达习惯。
- **开放协作**：欢迎《辐射》系列爱好者参与翻译、校对、考据和术语维护。

## 目录约定

- `original/`：保存原始资料，不应直接修改、移动、覆盖或重命名。
- `translation/`：保存中文译本；译本应保持源文件在 `original/` 下的相对路径、文件名和扩展名。
- `skills/van-buren-translation/`：保存项目专属翻译流程、风格指南、术语参考和检查脚本。
- `contributors.md`：记录参与翻译、校对、考据和维护的贡献者。
- `glossary.md`：维护项目级术语；存在多个合理译名时，应先讨论并确认。

### 路径映射示例

```text
原文：

译文：
translation/VanBurenTextSegregated10/Twin Mothers/PIPBOY_entries.txt
```

## 翻译原则

1. **先复制，后翻译**：将原文件按相对路径复制到 `translation/`，所有翻译只写入译本。
2. **保留原始格式**：不随意改变标题、段落、列表、表格、编号、脚本标签、变量和占位符。
3. **保护文档版式**：编辑 `.doc` 文件时，应保留图片、文本框、表格、页眉页脚和其他版式元素。
4. **检查文本编码**：处理 `.txt`、`.scc` 等文本文件前，先确认编码和换行格式，避免产生乱码。
5. **统一项目术语**：优先使用已确认译名；发现冲突或歧义时，不凭猜测自行决定。
6. **忠实而自然**：准确传达事实与设计意图，同时保持自然、克制且符合经典《辐射》气质的中文表达。

## 文档类型

| 类型 | 翻译重点 |
| --- | --- |
| 设计文档 | 保留玩法逻辑、任务条件、数值、变量和开发备注 |
| 背景设定 | 准确呈现时间、因果、组织关系和既定事实 |
| 对话文档 | 结合人物身份、关系、目的和情绪处理口吻 |
| 人物设计 | 区分客观设定、作者评价、角色自述和候选方案 |
